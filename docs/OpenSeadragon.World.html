setAutoRefigureSizes(valueopt)

As a performance optimization, setting this flag to false allows the bounds-change event handler on tiledImages to skip calculations on the world bounds. If a lot of images are going to be positioned in rapid succession, this is a good idea. When finished, setAutoRefigureSizes should be called with true or the system may behave oddly.
Parameters:
Name Type Attributes Description
value Boolean <optional>
 The value to which to set the flag.
